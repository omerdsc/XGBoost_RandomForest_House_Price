Veri Seti Tantm

Bu veri seti Trkiye'deki farkl ehirlerdeki ev fiyatlar hakknda bilgi ierir. Metrekare bykl, oda says, kat seviyesi, binann ya, stma tr ve bir evin fiyatn etkileyebilecek dier nitelikler gibi eitli zellikleri ierir.

  • Net_Metrekare: Evin net metrekare alan.

  • Brt_Metrekare: Evin brt metrekare alan.

  • Oda_Says: Evdeki oda says.

  • Bulunduu_Kat: Evin taban seviyesi.

  • Eya_Durumu: Evin eyal olup olmadn gsterir.

  • Binann_Ya: Binann ya (rnein; "0 (Yeni)", "5-10", "21 ve zeri").

  • Istma_Tipi: Istma tr (rn. "Kombi Doalgaz", "Klimal").

  • Fiyat: Evin Trk Liras (TRY) cinsinden fiyat. (Hedef)

  • ehir: Evin bulunduu ehir.

  • Binann_Kat_Says: Binann toplam kat says.

  • Kullanm_Durumu: Evin dolu ya da bo olduunu gsterir.

  • Yatrma_Uygunluk: Evin yatrma uygun olup olmad.

  • Site_erisinde: Evin bir konut kompleksi ierisinde olup olmad.

  • Takas: Evin baka bir gayrimenkul ile takas edilip edilemeyeceini gsterir.

  • Krediye_Uygunluk: Evin ipotek iin uygun olup olmad.

  • Tapu_Durumu: Mlkiyet unvannn hukuki durumu (rnein, "Kat Mlkiyeti").

  • Banyo_Says: Evdeki banyo says.

import re

def kat_donustur(kat):
    # Saysal katlar
    if re.match(r'^\d+\.Kat$', kat):
        return int(kat.split('.')[0])
    elif kat == '40+.Kat':
        return 40
    elif kat in ['Dz Giri (Zemin)', 'Yksek Giri']:
        return 0
    elif kat == 'Bahe Kat':
        return -1
    elif kat == 'Bodrum Kat':
        return -2
    elif kat in ['at Kat', 'at Dubleks']:
        return 99
    elif kat == 'Bahe Dublex':
        return -1
    elif 'Kot 1' in kat:
        return -1
    elif 'Kot 2' in kat:
        return -2
    elif 'Kot 3' in kat:
        return -3
    elif 'Kot 4' in kat:
        return -4
    else:
        return -999  # bilinmeyen/mstakil/villa gibi deerler

  • at Kat, at Dubleks 99 ile temsil edildi
  • Anlam belirli olmayan veya modellemede anlaml karl olmayan deerler (rnein Mstakil, Villa Tipi, Bilinmiyor) -999 olarak kodland
